A função RGBA tem o formato geral rgb(R, G, B, A) . Ela admite quatro parâmetros, que definem a proporção das cores RED, GREEN e BLUE que entram na composição da cor e o quarto parâmetro A (canal alpha da cor) que definine a opacidade. o valor maximo de cada decimal que corresponde sua cor é 255 e o valor minimo é 0
hue saturation e lightness HUE : Neste caso a escala vai de 0 a 360 que se baseia na cor que é o hue Saturação: vai de 0 a 100% 100% corresponde a cor mais forte 0% corresponde a mais fraca na escala de cinza. Lightness : 50% seria a cor mais pura possível e se for para o lado do 100% vai ficando mais claro e se for para o lado do 0% vai ficando o mais escuro possível.
